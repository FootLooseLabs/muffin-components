# muffin-components

The official component registry for the [muffin framework](https://github.com/FootLooseLabs/element). A curated collection of reusable `Muffin.DOMComponent` UI components.

- **[Using components](./USAGE.md)** — browse, search and add components to your project
- **[Contributing components](./CONTRIBUTING.md)** — publish a component to the registry
